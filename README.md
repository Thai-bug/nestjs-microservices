
## Description

Microservice with NestJS

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start [service-name]

# watch mode
$ npm run start:dev [service-name]

# production mode
$ npm run start:prod [service-name]
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
